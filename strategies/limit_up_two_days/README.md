# 限售股连续2个涨停策略

这是一个基于AKSHARE数据的量化交易策略，用于回测"连续2个涨停后T+1卖出"的策略。

## 策略说明

- **策略名称**: 连续2涨停T+1卖出
- **初始资金**: 100万元
- **回测周期**: 2025-01-01 至 2026-01-06
- **清仓日期**: 2026-01-07

### 策略逻辑

1. 扫描A股所有股票
2. 找出连续2个涨停（涨幅≥9.5%）的股票
3. 在连续涨停完成后的下一个交易日买入
4. 如果有多个这样的股票，按平均金额买入
5. T+1日全部卖出
6. 2026-01-07日清仓所有持仓

## 文件结构

- `config.py` - 策略配置参数
- `data_fetcher.py` - 数据获取模块（使用AKSHARE）
- `backtest_engine.py` - 回测引擎
- `main.py` - 主程序

## 使用方法

```bash
# 确保已安装依赖
pip install akshare pandas numpy

# 运行回测
python main.py
```

## 输出结果

回测完成后会输出：
- 最终资产和盈利情况
- 交易统计（买入/卖出次数）
- `trading_records.csv` - 详细交易记录
- `daily_values.csv` - 日净值曲线
